# Simple-Acccounting

简约记账 Android 应用，支持记录事项及金额，统计收入与支出。

## 功能

- 添加记账事项及金额
- 显示记账列表
- 统计总收入和总支出

## 技术栈

- Java
- SQLite 数据库
- Android Studio

## 使用说明

1. 克隆项目到本地  
2. 在 Android Studio 中打开项目  
3. 编译运行
